Getting Started

  1. Connect your wallet
  2. Click "New Build" and fill in your GitHub repo URL
  3. Select platform (Android/iOS/Both) and build type
  4. Confirm payment in THRON tokens
  5. Watch your build in real-time via WebSocket logs
  6. Download your APK/IPA when the build completes

Supported Source Types

  • GitHub — Public or private repos (with token)
  • GitLab — Public or private repos
  • ZIP Upload — Upload your source code directly

API Reference

POST /api/v1/builds — Submit new build
GET /api/v1/builds/:id — Get build status
GET /api/v1/builds/:id/logs — Get build logs
POST /api/v1/builds/:id/cancel — Cancel build
GET /api/v1/builds/:id/download/:platform — Download artifact
GET /api/v1/status/pricing — Get pricing
GET /api/v1/status/stats — Get system stats
